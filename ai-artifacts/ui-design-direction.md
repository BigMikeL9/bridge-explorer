# UI Design Direction

Bridge Explorer uses a calm, modern, data-first geospatial workspace style.

Design choices:
- Light neutral palette.
- Compact typography.
- Subtle borders and restrained shadows.
- Lucide icons for professional UI signals.
- Badges use color, icon, and text.
- Map-first/data-first layout with a right-side risk panel.
- Visible focus states for keyboard users.
- No emoji badges, loud gradients, or generic admin-dashboard styling.
