# Prompts Used

The implementation followed phase-based prompts for a Mach9 take-home assessment named Bridge Explorer.

Key prompt themes:
- Build incrementally and stop after each phase.
- Prefer consistency, maintainability, and readability.
- Keep database schema, API DTOs, and UI state flat.
- Isolate raw FHWA/NBI names to importer code.
- Support Pennsylvania-only and national all-states datasets from the start.
- Avoid auth, editing, exports, saved searches, AI chat, and extra dashboards.
